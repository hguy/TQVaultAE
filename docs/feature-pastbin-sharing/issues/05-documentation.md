## Acceptance criteria

- [ ] Documentation covers PasteBin export/import with API key setup instructions
- [ ] Documentation includes a link to https://pastebin.com/doc_api for obtaining an API key
- [ ] Documentation covers the full vault import strategy (Replace / Create New / Cancel)
- [ ] Documentation file is at ./documentation/IMPORT_EXPORT.md
